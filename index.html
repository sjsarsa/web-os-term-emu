Full Linux OS (Alpine) and terminal running entirely in your browser.
Default shell is bash. Only basic utilities available, no network or GUI.

Copy and paste works with Ctrl+Alt+C and Ctrl+Alt+V.

The linux state is autosaved to your browser storage once every minute. The base state requires roughly 80MB of space.

Powered by V86, Alpine Linux, and xterm.js.
